# 🧵 space_app_api

## ℹ️ Description

A test focused api with jest for making tests before going to space ;) ... This was heavily influenced on learning deeper features of jest such as using matchers and writing my own matchers

## 🔮 Tech Stack

- JavaScript
- Express
- Jest
- Pug
- Morgan

## 📦 Installation

```bash
npm install
```

## 🚀 Running the app

```bash
# development
npm start

# test
npm run test

# test watch
npm run test:watch

```
